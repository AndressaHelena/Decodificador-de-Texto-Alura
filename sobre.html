Regras Codificador:

"e" é convertido para "enter"

"i" é convertido para "imes"

"a" é convertido para "ai"

"o" é convertido para "ober"

"u" é convertido para "ufat"

Apenas letras minúsculas

Não permite acentuação

Não permite números
